# oshea's Plugin Index

This repository is a registry index for non-bundled [oshea](https://github.com/brege/oshea) plugins.

## Installing

Install plugins directly from each plugin's repository.

```bash
oshea plugin add https://github.com/brege/restaurant-menu
```

## Index

- **`d3-histogram-slide`** 路 
[github.com/brege/d3-histogram-slide](https://github.com/brege/d3-histogram-slide)    
Presentation slides with d3.js histograms and LaTeX equations.

- **`gh-social-preview`** 路 
[github.com/brege/gh-social-preview](https://github.com/brege/gh-social-preview)  
A GitHub social preview card.

- **`hierarchy-table`** 路 
[github.com/brege/hierarchy-table](https://github.com/brege/hierarchy-table)  
A table as a presentation slide.

- **`restaurant-menu`** 路 
[github.com/brege/restaurant-menu](https://github.com/brege/restaurant-menu)  
A single page restaurant specials menu.
